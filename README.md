# MMC Frosty Modding Tools

Modding tools for Madden Mod Community (MMC) built on the Frosty toolset.

## Download

Grab the latest packaged build from the [Releases](https://github.com/bphit4/MMC-Frosty-Modding-Tools/releases) page.

- **v1.1.0.1** — `MMC_Modding_Tools_v1.1.0.1.rar`

## Installation

1. Download the release archive.
2. Extract it with a tool that supports RAR archives (e.g. WinRAR or 7-Zip).
3. Run the included tools.

The College Football Modding Community DOES NOT allow the use of any mods during online play, including dynasty mode. Follow these steps carefully to ensure proper setup for College Football 27 modding.
## 1. Initial Setup

### 1.1 Extract the Rar File

Extract the contents of the `MMC_Modding_Tools_v1.1.0.0` file using WinRar, WinZip, or any decompression tool, and place them in a dedicated folder on the same drive as your College Football 27 installation.

### 1.2 Disable Original Anti-Cheat

Navigate to your College Football 27 game folder:

`C:\Program Files\EA Games\EA SPORTS College Football 27`

Or you can right click the CFB 27 shortcut exe and select open file location.

Rename:

`EAAntiCheat.GameServiceLauncher.exe`

to something else, such as:

`_EAAntiCheat.GameServiceLauncher.exe`

You can call it whatever you want.

### 1.3 Replace Anti-Cheat Launcher

Copy the `EAAntiCheat.GameServiceLauncher.exe` from the extracted `MMC_Modding_Tools` folder and paste it into your College Football 27 game folder, replacing the modified version.

## 2. MMC Mod Manager is to play with mods

### 2.1 Setup MMC Mod Manager

Run MMC Mod Manager as an administrator:

Right-click the application, select `Properties > Compatibility > Run as administrator`.

Launch the MMC Mod Manager by opening the exe located in your mod folder.

### 2.2 Importing and Activating Mods

Open MMC Mod Manager and select `Add Mod`.

Navigate to the location where your `.fbmod` files are saved and import them.

Once imported, click on each mod you want to use, and then click `Apply`.

### 2.3 Launching the Game

With your mods applied, click the `Launch` button within MMC Mod Manager to start College Football 27 with the selected mods active.

If you run into issues, try using the `Delete ModData and Launch` button.

This will not affect your mods or saves, but will rebuild the modded files from scratch, which can sometimes help resolve random problems.

## 3. MMC Editor is to create your own mods

### 3.1 Setting Up MMC Editor

Run MMC Editor as an administrator:

Right-click the application, select `Properties > Compatibility > Run as administrator`.

Launch the MMC Editor by opening the exe located in your mod folder.

### 3.2 Creating or Editing Mods

Open MMC Editor, navigate to the necessary files you want to modify, and make your desired changes.

Save your modifications and ensure they are properly configured before launching the game.

## 4. Restoring Original Files, if needed

### 4.1 EAAntiCheat.GameServiceLauncher.exe

If you need to restore the original anti-cheat launcher to play online, remove the modified version and rename:

`_EAAntiCheat.GameServiceLauncher.exe`

back to its original name in the College Football 27 game folder.

If `CryptBase.dll` is present in the game folder, make sure to delete it otherwise you will see crash handler report pop up.
